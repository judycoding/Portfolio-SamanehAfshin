Space to Place :

Developing a sense of place through digital storytelling is the main goal in this project. Considering a critical distinction between “space” and “place” is vital to upgrade new opportunities for visitors to interact with a place. The game is an application mobile quipped with location-aware system which run the game only when the user is in a specific location.

Mission :

Sightseeng mission : There is a map with different spots in grey color. Visitor should color the map by going to the place indicated in the map and download the AR video they find in the place

hunting mission : A surprising hunting mission is presented in every spot which is shown in the app when the user is physically present in the place. This hunting mission is related to a historical or cultural point of the city.
