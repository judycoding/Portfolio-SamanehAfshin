CoTour
Urban Game

Keywords : Mobile storytelling, Place, Space, location-based game, city exploration, Memento.

Space to Place : Developing a sense of place through digital storytelling is the main goal in this project. Considering a critical distinction between “space” and “place” is vital to upgrade new opportunities for visitors to interact with a place. The game is an application mobile quipped with location-aware system which run the game only when the user is in a specific location.

Coffe House

CoffeeHouse Business Scenario : CoffeeHouse CoffeeHouse è una catena internazionale con sedi a Milano, Bologna, Torino, Berlino e altre città. Il suo obiettivo è servire caffè di qualità con un tocco locale. Le bevande hanno un prezzo compreso tra 3 e 8 dollari. Le bevande speciali sono preparate al momento al bancone. Per ciascuna delle città in cui operano attualmente, hanno creato un'esclusiva bevanda a base di caffè che può essere ordinata solo nella loro città. Questo ha ispirato i fan a cercare i negozi CoffeeHouse in altre città per provare tutte le creazioni esclusive.
